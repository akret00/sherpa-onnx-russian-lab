# КОНТЕКСТ ПРОЕКТА: SHERPA-ONNX-RUSSIAN-LAB

## АРХИТЕКТУРА И СТЕК
- Python 3.10+, СТРОГО без Torch, Torchaudio и Librosa.
- Пайплайн: Аудио/Микрофон -> FFmpeg (через subprocess) -> NumPy-буфер (16kHz, 16-bit PCM Mono) -> sherpa-onnx.
- Модели: Silero VAD (активность) + GigaAM v3 / Qwen3-ASR (распознавание) + 3Speaker (диаризация).

## СТРУКТУРА
- /src/ - исходный код (основная логика).
- /src/benchmark/ - скрипты бенчмарков, расчеты показателей WER/DER (в процессе разработки, пока есть только скрипты для разметки).
- /dataset/ - разметка аудиофайлов (файлы разметки публичные, исходные аудиофайлы приватные, выкладываться не будут).
- /docs/ - отчеты об экспериментах (могут отставать от кода, код приоритетнее).
- win_asr_setup.bat/.ps1 - автоматическое развертывание среды для Windows.
- config.yaml - главный конфигурационный файл.

## ОСНОВНЫЕ СКРИПТЫ
- src/asr_vad.py - входящая точка для распознавания речи из файла или микрофона. Работает потоково сегмент за сегментом.
- src/asr_vad_diariz_centr.py - входящая точка для распознавания и диаризации речи из файла или микрофона. Работает потоково сегмент за сегментом.

## ПРАВИЛА РАЗРАБОТКИ ДЛЯ ИИ
- Обязательная типизация (Type Hinting) для всех функций и атрибутов классов.
- Докстринги короткие: 1-2 строки, объясняющие ЧТО это делает, без длинных блоков Args/Returns.
- Новые фичи писать максимально легковесно, без добавления тяжелых библиотек. Желательно оставаться на numpy, pyyaml, sherpa-onnx.
