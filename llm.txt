# КОНТЕКСТ ПРОЕКТА: SHERPA-ONNX-RUSSIAN-LAB

## АРХИТЕКТУРА И СТЕК
- Python 3.10+, СТРОГО без Torch, Torchaudio и Librosa. Инференс — только CPU.
- Пайплайн: Аудио/Микрофон -> FFmpeg (subprocess) -> NumPy-буфер (16kHz, 16-bit PCM Mono) -> sherpa-onnx.
- Модели: Silero VAD + GigaAM v3 / Qwen3-ASR + 3Speaker + Pyannote Segmentation.
- Данные: YAML-формат (без RTTM/JSON), основная модель — dataclass `AudioSegment`.

## СТРУКТУРА
- /src/ - основные компоненты и фабрика пайплайнов.
- /src/benchmark/ - скрипты датасета, движок, расчет WER/CER, YAML-метрики.
- /dataset/ - разметка, сценарии, генерация сцен.
- /docs/ - отчеты и ТЗ (код приоритетнее).
- config.yaml - конфигурация.

## ОСНОВНЫЕ КОМПОНЕНТЫ И СКРИПТЫ
- VAD с Оракул-режимом (из разметки).
- Пайплайны: `VAD + ASR`, `VAD + ASR + Diarization` (косинусное сходство), `Pyannote + ASR`.
- Датасет: автоматизация от черновой разметки до генерации сцен и тестовых аудиофайлов.
- Бенчмарк: автоматизация от спецификации до метрик.

## ПРАВИЛА РАЗРАБОТКИ ДЛЯ ИИ
- СТРОГАЯ типизация (mypy).
- Докстринги короткие: 1-2 строки, объясняющие ЧТО это делает, без длинных блоков Args/Returns.
- Легковесность: без тяжелых библиотек, алгоритмы метрик на чистом Python/NumPy.
