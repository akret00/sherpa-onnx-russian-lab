# Sherpa-ONNX Russian Lab
Сборник практических экспериментов по распознаванию (ASR), диаризации и синтезу русской речи на базе библиотеки `sherpa-onnx` для Python.

> **Личный полигон для тестов**: код предоставляется "как есть", без идеальной архитектуры и поддержки.

## Цели проекта
- Сборка работающих примеров ASR (распознавание речи) для русского языка.
- Тестирование разных моделей ASR и вариантов диаризации (разделение по ролям).
- Создание скриптов для формирования протоколов совещаний.
- Голосовое взаимодействие с LLM-агентами (в перспективе).

---

## Особенности подхода
Проект строится на принципе **максимальной легкости** и минимального количества зависимостей:
- **Минимализм**: Полный отказ от тяжелых зависимостей вроде `torch`, `torchaudio` или `librosa`. 
- **Легкий стек**: Только `sherpa-onnx`, `numpy` и `ffmpeg`. 
- **Поточная обработка**: Чтение аудио напрямую из `stdout` FFmpeg через `subprocess` в `numpy`-буфер.
- **Скорость**: Инференс на ONNX Runtime позволяет работать с приемлемой производительностью на CPU офисных ноутбуков без дискретных видеокарт.
- **Простота**: Скрипты не требуют настройки сложных окружений и огромных объемов RAM.

---

## Технологический стек
- **Движок**: `sherpa-onnx` (ONNX Runtime)
- **Язык**: Python 3.10+
- **Обработка звука**: FFmpeg (через стандартный `subprocess`)
- **Математика**: `numpy`
- **Железо**: Обычные офисные ноутбуки и ПК с современными CPU. Наличие GPU не требуется.

---

## Используемые модели:
- **ASR (Речь в текст)**: 
  - `nemo-ctc-punct-giga-am-v3-russian-int8` (отличная пунктуация)
  - `qwen2.5-asr-0.6B-int8` (высокая точность)
- **VAD (Детекция тишины)**: `silero_vad.onnx`
- **Speaker ID (Эмбеддинги)**: `3dspeaker_speech_eres2net_base_sv_zh-cn_3dspeaker_16k.onnx`
- **Segmentation**: `sherpa-onnx-pyannote-segmentation-3-0`

## Эксперименты в работе

### 1. Диаризация длинных записей через VAD и центроиды
Разделение спикеров путем накопления качественных эмбеддингов и расчета центроидов в реальном времени. Без использования тяжелых сегментаторов.
*   *Статус: В разработке (рефакторинг).*

### 2. Диаризация через Pyannote Segmentation
Классический подход с сегментацией потока на блоки по 20-30 минут с последующей склейкой текста и ролей.
*   *Статус: Эксперименты с логикой склейки.*

### 3. Протоколирование совещаний
Сквозной сценарий: превращение аудиозаписи в структурированный текстовый протокол с разделением по именам спикеров.

## Как запустить
1. Необходим Python 3.10+
2. Установите зависимости:
   ```bash
   pip install sherpa-onnx numpy
   ```
3. Убедитесь, что в системе установлен `ffmpeg`.
4. Скрипты находятся в соответствующих папках в `/experiments`.
